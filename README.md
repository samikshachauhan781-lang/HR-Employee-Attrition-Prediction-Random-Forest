# HR Employee Attrition Prediction

## 📌 Project Overview
This project is a Machine Learning classification model that predicts whether an employee will leave the company or not based on various workplace and personal factors.

---

## 📊 Problem Statement
Employee attrition is a major issue for companies. This model helps predict which employees are likely to leave so that companies can take preventive actions.

---

## 🤖 Machine Learning Model Used
- Random Forest Classifier

---

## 📂 Features Used
- Salary  
- Job Satisfaction  
- Work Environment  
- Performance Rating  
- Work-Life Balance  

---

## 🎯 Output
- 0 → Employee will stay  
- 1 → Employee will leave  

---

## ⚙️ Technologies Used
- Python  
- Pandas  
- NumPy  
- Scikit-learn  

---

## 🚀 Purpose
To help HR departments reduce employee turnover using data-driven predictions.
